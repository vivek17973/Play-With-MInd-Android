Play With Mind - Earn Rewards

This is a starter Android Studio project with:
- Firebase Phone OTP Authentication
- Placeholder for Sudoku game
- AdMob integration (add initialization in MainActivity)
- Wallet and reward logic hooks
- Referral, leaderboard, streak logic left to implement in backend

Build instructions:
1. Open this project in Android Studio.
2. Sync Gradle (internet required).
3. Run: ./gradlew assembleDebug
4. Installed APK at: app/build/outputs/apk/debug/app-debug.apk
